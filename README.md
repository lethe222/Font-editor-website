# 字体裁剪工具

一个在线字体裁剪小工具。上传字体后，输入或勾选需要保留的字符，工具会在本地生成裁剪后的字体文件，并支持自行选择导出格式。

## 功能

- 支持上传 `TTF`、`OTF`、`WOFF` 字体
- 支持手动输入字符
- 支持一键勾选常用字符集
- 实时预览裁剪后的字体效果
- 预估裁剪后文件大小
- 最终可导出为 `OTF`、`TTF` 或 `WOFF`
- 所有处理都在浏览器本地完成，不会上传服务器

## 运行方式

```bash
npm i
npm run dev
```

开发服务器启动后，浏览器打开终端提示的地址即可使用。

## 构建

```bash
npm run build
```

## 使用说明

1. 打开页面后上传一个字体文件。
2. 在输入框中填写需要保留的文字，或者勾选下方预设字符集。
3. 右侧会显示裁剪后的字符数量和预估大小。
4. 点击下载按钮，即可导出裁剪后的字体文件。

## 支持格式

- 输入：`TTF`、`OTF`、`WOFF`
- 输出：`OTF`、`TTF`

## 说明

- 本项目是一个前端字体工具，适合做网页字体子集化和轻量字体导出。
- 若浏览器标签页图标没有立即更新，可以尝试强制刷新或清理缓存后再看。
